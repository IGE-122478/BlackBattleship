# BlackBattleship

Projeto de testes de caixa-preta (Black-Box Testing) com Selenium WebDriver
para validação funcional do jogo de Batalha Naval online.

## Tecnologias utilizadas

- **Selenium WebDriver 4** — automação de browsers
- **JUnit 5** — framework de testes
- **Maven** — gestão de dependências
- **Page Object Model (POM)** — padrão de desenho

## Estrutura do projeto

- `src/main/java` — Page Objects (classes que representam páginas web)
- `src/test/java` — Classes de teste
- `reports/` — Relatórios de execução de testes
